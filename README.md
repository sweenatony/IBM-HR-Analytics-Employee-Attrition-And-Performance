# IBM-HR-Analytics-Employee-Attrition-And-Performance

📌 Project Overview

This project analyzes employee attrition using the IBM HR dataset to uncover the key factors influencing employee turnover.

The goal is to help organizations reduce attrition, improve employee satisfaction, and make better HR decisions.

🎯 Business Problem

Employee attrition leads to:

Increased hiring and training costs

Loss of experienced employees

Reduced productivity

This project answers:
👉 Why are employees leaving?
👉 What factors influence attrition the most?

🎯 Objectives

Understand key drivers of employee attrition

Analyze employee behavior and patterns

Provide actionable insights for HR decision-making

🛠️ Tech Stack

Python – Data Cleaning & EDA

Pandas, NumPy – Data Manipulation

Matplotlib, Seaborn – Data Visualization

Power BI – Interactive Dashboard

Excel – Initial Data Exploration

📂 Dataset

IBM HR Analytics Dataset (Fictional dataset created by IBM data scientists)

Contains 1470 employees and 30+ features

🔍 Key Insights

Employees working overtime have ~3x higher attrition

Low salary groups show the highest attrition

Entry-level employees are more likely to leave

Sales roles experience the highest attrition

Employees with poor work-life balance tend to leave more

Single employees have higher attrition than married employees

Younger employees (20–40) switch jobs more frequently

Longer commute distance increases attrition

Frequent business travel leads to higher attrition

Hourly rate and monthly rate have minimal impact on attrition

📊 Dashboard

The Power BI dashboard provides an interactive view of:

Attrition by Age, Salary, Job Role, and Department

Work-Life Balance & Overtime impact

Employee demographics and behavior patterns

🧠 Approach

Data Cleaning & Preprocessing

Removed irrelevant columns

Checked for missing values and duplicates

Exploratory Data Analysis (EDA)

Analyzed relationships between variables

Identified trends and patterns

Visualization

Created meaningful plots using Python

Built an interactive dashboard in Power BI

🚀 Business Recommendations

Reduce overtime workload to improve retention

Improve salary structure for lower-level employees

Focus on work-life balance initiatives

Provide career growth opportunities for entry-level roles

Reduce employee stress in high-pressure roles like Sales

🚀 Conclusion

This project provides a comprehensive analysis of employee attrition using the IBM HR dataset. The findings highlight that factors such as overtime, low salary, poor work-life balance, and job role play a significant role in employee turnover.

The analysis demonstrates how data can be used to uncover hidden patterns in workforce behavior and support data-driven HR decision-making. By addressing these key factors, organizations can improve employee satisfaction, reduce attrition, and enhance overall productivity.

Overall, this project reflects my ability to perform end-to-end data analysis, from data cleaning and exploration to visualization and insight generation, using tools like Python and Power BI.
